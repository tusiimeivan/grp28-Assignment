##  Project Description

This project is a web-based e-commerce application developed using HTML, CSS, and JavaScript. It allows users to browse products, search and filter items by category, add products to a shopping cart, and proceed to checkout.

The system demonstrates key JavaScript concepts such as DOM manipulation, event handling, arrays, and dynamic content rendering. It provides a simple and interactive interface for managing an online shopping experience.

Key features include:
- Product listing
- Search and category filtering
- Add to cart functionality
- Cart management (view and remove items)
- Total price calculation
- Checkout form for capturing user details

## ⚙️ How to Run the Project

1. Download or clone this repository  
2. Open the project folder  
3. Double-click on `index.html` or open it in a web browser  
4. Use the navigation links to access Cart and Checkout pages
